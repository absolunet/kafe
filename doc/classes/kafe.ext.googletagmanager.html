kafe.ext.googletagmanager

Class

Version 0.1.0

Extra methods for the Google Tag Manager. Requires GTM to be included

Source: dist/ext/googletagmanager.js

Methods

addCheckoutOptions

@110

Add named checkout options to the list of possible options to be triggered, with the possibility of replacement tokens. Default tokens provided: - {{uri}}: Current pathname

addCheckoutOptions
  • checkoutOptions

Parameters:

  • checkoutOptions Object

    List of possible checkout options and their config

    • step String

      The step value.

    • option String

      The option value.

Example:

// Create named checkout options
					kafe.ext.googletagmanager.addCheckoutOptions({
					    'checkout-type': {
					        step:  'Identification',
					        label: 'Identified as {{type}}'
					    },
					    'checkout-shipping': {
					        step:  'Shipping method',
					        label: 'Shipped via {{method}}'
					    }
					});

addEvents

@61

Add named events to the list of possible event to be triggered, with the possibility of replacement tokens. Default tokens provided: - {{uri}}: Current pathname

addEvents
  • events

Parameters:

  • events Object

    List of possible events and their config

    • category String

      The eventCategory value.

    • action String

      The eventAction value.

    • label String

      The eventLabel value.

Example:

// Create named events
					kafe.ext.googletagmanager.addEvents({
					    'newsletter-subscribed': {
					        category: 'Subscription',
					        action:   'Newsletter',
					        label:    '{{uri}}'
					    },
					    'product-addedtocart': {
					        category: 'Cart',
					        action:   'Product added - {{uri}}',
					        label:    '{{productname}}'
					    }
					});
