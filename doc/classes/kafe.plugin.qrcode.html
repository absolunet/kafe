kafe.plugin.qrcode

Class

Version 0.1.1

Returns a HTML table for displaying a QR code, encoded from the sent string.

Source: build/kafe/plugin/qrcode.js Unit test: Report

Methods

generate

@20

Generate the HTML table of the QR code.

generate
  • text
  • options
String

Parameters:

  • text String

    The text to convert to a QR code.

  • [options] Object Optional

    Additional options.

    • [version = 4] Number Optional

      QR code version (1-40) [Explanation on Wikipedia].

    • [correctLevel = 'H'] String Optional

      QR code correction level (L,M,Q,H) [Explanation on Wikipedia].

Returns:

String

The HTML table representing the QR code.

Example:

kafe.plugin.qrcode.generate('Scan me beautiful');
// returns "<table><tbody><tr><td class="X"></td><td></td><td class="X"></td></tr>[...]</tbody></table>"
