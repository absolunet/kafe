kafe.style

kafe.style.quickMenu( [params] )

Adds a hover behavior to an html list to show/hide sub-items with custom speeds and delays.

  • Parameters :
    • selector ( string ) : The container element. The behavior will be attached to the children elements.
    • animation ( string ) : Animation style. Possible values are 'fade' and 'slide'. Default value is 'slide'.
    • openSpeed ( number ) : Speed of the opening animation in milliseconds. Default value is 200.
    • openDelay ( number ) : Delay before the opening animation in milliseconds after entering. Default value is 500.
    • closeSpeed ( number ) : Speed of the closing animation in milliseconds. Default value is 150.
    • closeDelay ( number ) : Delay before the closing animation in milliseconds after leaving. Default value is 400.

    selector: '.quickMenuDemo ul', animation: 'fade',
    openSpeed: 200, openDelay: 400,
