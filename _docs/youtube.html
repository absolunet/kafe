kafe.ext.youtube

kafe.ext.youtube.search( query, callback )

  • Searches youtube with the specified query then forwards the results as the first argument to the callback.
  • There are 3 ways to search that can be conbined if needed :
    • q ( string ) : Simple keywords query, works the same way as the search field on YouTube.com.
    • author ( string ) : Only return results from a specific author.
    • category ( array [ string ] ) : Try and find specific terms in the tags of the videos. (Not very effective right now...)
kafe.ext.youtube.search({ query: 'Gallium Pro', author: 'Argon18bikes' }, function( results ) {
	//Parse the simple results.
});
