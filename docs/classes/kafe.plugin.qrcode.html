String generate( text, [options] )
builds/kafe/plugin/qrcode.js:20

Generate the HTML table of the QR code.

Parameters:
  • text <String>

    The text to convert to a QR code.

  • [options] <Object>

    Additional options.

    • version <Number>

      QR code version (1-40) [Explanation on Wikipedia].

    • correctLevel <String>

      QR code correction level (L,M,Q,H) [Explanation on Wikipedia].


Returns: <String>

The HTML table representing the QR code.

Example
kafe.plugin.qrcode.generate('Scan me beautiful');
// returns "<table><tbody><tr><td class="X"></td><td></td><td class="X"></td></tr>[...]</tbody></table>"
