kafe.ext.bbq.getHashbang()

  • Returns string
var obj = kafe.ext.bbq.getHashbang();

kafe.ext.bbq.getHashbangParams()

  • Returns object
var obj = kafe.ext.bbq.getHashbangParams();

kafe.ext.bbq.setHashbang( [params], [mergemode] )

  • Set a new param
  • Modify existing params
  • Set from string

Merge Modes :

  • 0 (default) : Override existing parameters.
  • 1 : Preserve existing parameters.
  • 2 : Override hashbang completely.
kafe.ext.bbq.setHashbang({ right: '4', now: '5' });
kafe.ext.bbq.setHashbang({ mange: '111', mon: '222', pain: '333' });
kafe.ext.bbq.setHashbang('/MangeMonPain/RightNow', 2);
